# 架构决策记录（ADR）

每条决策一个文件，命名建议：`NNN-简短标题.md`（`NNN` 为三位序号）。

## 何时写 ADR

- 选定语言/框架/数据库/部署形态
- 引入或弃用重要中间件
- 安全、合规、多租户等边界变化
