# 任务单

每个可执行、可交接的任务一个文件，命名建议：`NNN-简短标题.md`。

## 使用方式

1. 新建任务时复制 [`_template.md`](_template.md)。
2. 开发过程中更新「进度」「已修改文件」。
3. 结束任务时填写「交接给下一会话」。
