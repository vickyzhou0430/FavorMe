# 点醒 · 设计方案

## 三个风格方向

**方向一：禅意水墨**
以中国水墨画的留白与晕染为美学基础，用墨色渐变和宣纸质感传递"内心澄明"的意境。概率：0.07

**方向二：深夜诊室**
深色背景，冷白字体，像一间安静的心理咨询室。用克制的排版和微弱的光晕营造私密感与信任感。概率：0.04

**方向三：纸质温暖**
米白底色，手写感衬线字体，柔和的暖黄光晕，像一本翻开的日记本。亲切、温暖、没有压迫感。概率：0.06

---

## 选定方案：深夜诊室（Dark Therapy Room）

### Design Movement
Late-night Brutalism meets Therapeutic Minimalism — 克制的黑暗美学，用极简排版制造情绪张力。

### Core Principles
1. **克制即力量**：界面元素极少，每一个元素都有明确意图，没有装饰性噪音
2. **文字是主角**：所有视觉重心都在文字上，字体大小和粗细的对比制造层次
3. **动效传递情绪**：过渡动画不是装饰，而是"正在思考"的可视化表达
4. **移动端优先**：H5场景，所有设计决策以375px宽度为基准

### Color Philosophy
深色背景（#0D0D0F）营造私密感，像深夜独处的房间。唯一的强调色是一种暖琥珀色（#E8A84C），代表"点亮"的那一刻。文字用近白（#F0EDE8）而非纯白，避免刺眼。

### Layout Paradigm
单列全屏流式布局，每个阶段占满整个视口高度。用垂直滑动的方式在"倾诉"→"测试"→"报告"三个阶段之间切换，而非页面跳转。

### Signature Elements
- 琥珀色光晕：在关键操作按钮和点醒金句上出现，代表"被照亮"
- 细线分隔：用0.5px的细线而非色块来分隔内容区域
- 打字机动效：AI的问题用逐字显示的方式出现，模拟"正在思考"

### Interaction Philosophy
每一次交互都应该有轻微的物理反馈感。选项卡片按下时有微小的位移，选中后有柔和的高亮过渡。整个过程应该感觉像是在和一个安静、专注的人对话。

### Animation
- 页面切换：200ms ease-out 向上滑入
- 文字出现：逐字打字机效果，每字间隔30ms
- 选项卡片：hover时轻微上移2px，active时scale(0.98)
- 结果页：金句先淡入，然后分析文字逐段出现

### Typography System
- 标题/金句：Noto Serif SC（衬线），传递庄重感
- 正文/选项：Noto Sans SC（无衬线），清晰易读
- 字号层级：金句48px → 题目22px → 选项16px → 辅助文字13px

### Brand Essence
点醒是一面镜子，为纠结中的年轻人照见自己。克制、深邃、直击内心。

### Brand Voice
不说教，不给答案，只是帮你听清自己内心的声音。
示例标题："你已经知道答案了，只是还没敢承认。"
示例CTA："开始测试"而非"立即使用"

### Signature Brand Color
琥珀金 #E8A84C — 黑暗中的那一点光。
